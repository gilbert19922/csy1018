Alignment Is the term for the layout of text or pictures on the webpage.
Attribute Is a feature that give more detail to an element for example image – alt tags.
Box Model Is a predefined setup for a webpage that encases every element.
Class The method of naming elements.
CMYK colour model. Stands for cyan, magenta, yellow, and black. This set of colors is used in print design because of the way paper absorbs light.
Colour Schemes. Are a set of colours that you are using consistently throughout a design.
Commenting Is the way of detailing what things do in your CSS or HTML file so anyone else who looks at it can easily determine what the code does.
Colour Wheel Is the circle of colours showing the relationship between colours from primary to tertiary.
Complementary Colours. Is the colours that we are on opposite sides of the colour wheel.
Compression. The elimination of excess data, this is quite useful with image files more than text files.
Containers Are a way of making content appear on the webpage without them websites would not exist
Contrast. the difference between colours, shapes and design elements.
Crop. Is a way to reframe the image or resize the image ratio.
CSS. (Cascading Style Sheet) Is a page of code separate from the HTML documents that tell the HTML how to look when someone opens the page.
DPI. Dots per Inch refer to the resolution of a printed object, the greater the resolution the better the quality
Element An element is any part of the webpage that is enclosed by a set of tags with attributes.
Feathering A technique to smooth out edges of features.
Flex Is a CSS display property that causes the elements to become a flex conainter. Flex containers cascade down from parents to child elements.
File Path Is the location of the saved documents in terms of websites they follow file paths to call objects on the page like videos or pictures and the CSS.
Font A specified style and size. A good example of this would be Comic Sans size 15.
Grid The theoretical map of horizontal and vertical lines that align images and text in a HTML document.
Hex Code The code used within HTML and CSS to designate a specific colour using the # key followed by the number to make it the correct.
HTML The language used to display content on the web we are now on version 5.
Hypertext Is the words that contain links to other text, rather than a flow of text
Kerning Is the space between the letters, this can be applied separately however some fonts include kerning as a part of their font style.
Lossless Is a form of compression allows the detail of the image to be restored when editing the image.
Lossy A data compression technique that loses detail as the file size is decreased.
Negative space The empty space around a webpage or image.
Open Fonts Are the current standard for font formats. They contain both screen and print versions allowing for easy printing of webpages. And are compatible with Windows, Mac and Linux.
San-Serif The lack of a small line attached to the end of a stroke.
Serif A small line attached to the end of a stroke.
Syntax Is the specific combination of characters making up working code. It is different in each language for example HTML uses ords to generate your tags where as CSS uses “{words}” to generate the correct styling.
Tags These are used in HTML documents to create the elements some are predefined with set styles that can be edited within CSS for example: H1, H2, H3, H4, H5 and H6 are all heading tags and P is a paragraph tag these have predefined styles.
Validator Is a program that checks your documents for syntax errors within the code. They apply to any format and language but are commonly used for checking HTML and CSS.
White Space The blank space surrounding an object in design is always white hence the name white space, compared to negative space which can be any other colour
Vector image Instead of using pixels to represent images vector ones use lines and curve paths to generate the image. Due to not relying on pixels scaled images maintain image clarity and quality.
Raster image Is an image file that has a grid of pixels rather than the vector lines to generate the image. They are commonly smaller than vector images as they can be compressed using lossy compression
Pixel Is the smallest measurement of an element on a computer.
Resolution A way of measuring the level of detail in an image. The higher the resolution the higher the file size will be as it the larger resolution stores more detail.
Alt Tags Are a way of describing what is on a video or image to allow screen readers to detect what is there and describe it to a person who is blind.
JavaScript Is a language used to program dynamic items onto a webpage it has its own syntax and is often server-side rather than client side.
Web Standards These are set of rules that have been introduced by international bodies to define how the Web works. For example W3C.
404 Page These are error pages that appear when the server you are trying to reach cannot find the source that should be navigated to, some companies decide to do funny pages – my favourite ones are the south park ones
Mozzila Developer This is a site that contains a lot of useful information for new and experienced web developers, it holds up to date information as the web changes for example from XHTML to HTML5 it changed
W3C Defines how mark-up language, HTML, and style definitions, CSS, work.
Accessibilty Refers to best practices for keeping a site usable if someone has physical or technical restrictions. For example screen readers for blind people, and resource low pages for people with slow internet connections.
Domain Is the name of the website link people use to go to a website. It is linked through to a file within the directory, the first page is good practice to be called index
Wireframes These are the initial designs for the layout of the website you are going to build. They focus on basic layout without using too much content.
Hyperlink A link on a page that takes you to another document. Could be an image, a sound file or another web page
Developer Forums These are good pages that have lots of web developers on who will help each other solve problems they have. Stack Overflow is my preferred go to forum as it has both web development as well as a lot of other variation on there
